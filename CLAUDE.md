# CLAUDE.md

## Project Overview

一般社団法人ひなたぼっこ倶楽部の公式ウェブサイト。

## Design Guidelines

プロフェッショナルで一流のWEBデザイナーとして制作・レビューすること。

- 情報の優先順位を明確にする
- タイポグラフィのヒエラルキーを意識する
- 余白のリズムを統一する（8pxグリッド）
- ブラウザ互換性を確保する（Safari, Chrome, Brave等）
- モバイルファーストでレスポンシブに対応する
