# Website

Static website hosted on GitHub Pages.

## Structure

```
├── index.html      # Main page
├── styles.css      # Stylesheet
├── script.js       # JavaScript
└── images/         # Image assets
```

## Development

```bash
# Local preview
python3 -m http.server 8000
```

## Deployment

Automatically deployed via GitHub Pages on push to `main` branch.
